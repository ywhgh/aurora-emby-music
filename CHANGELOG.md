# Changelog

## v0.93.85

- 修复音乐桥搜索结果音质显示不真实的问题，移除按平台名称猜测 `SQ`、`MP3`、`1080P` 的兜底逻辑。
- 搜索结果会后台解析音源桥 `/media`，仅在拿到真实播放地址元数据后显示 `FLAC`、`SQ`、`320k`、`MV 720P` 等音质标签。
- 新增 `检测中`、`源站未标注`、`MV 未标注` 状态，解析不到真实字段时不再伪造音质。
- 本地音乐通过音源桥返回真实文件格式；存在 `ffprobe` 时进一步读取 codec 和 bitrate。
- 加固外部搜索结果合并逻辑，避免旧数据覆盖新解析出的真实音质。
- 同步版本与缓存到 `0.93.85`。

### 验证

```powershell
npm run check
```
