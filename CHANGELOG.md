# Changelog

## v0.92.92

- 手机端搜索建议改为固定视口面板，避免跟随顶部栏容器裁切或撑出屏幕。
- 手机搜索结果保留快捷操作按钮，可直接下一首播放、加入队列、播放专辑/歌单。
- 收紧搜索建议行距、封面和图标尺寸，小屏下更容易扫读和点击。
- 主页顶部改为欢迎区与媒体统计并排布局，释放下方音乐内容的垂直空间。
- 7 个媒体统计块升级为紧凑白色小卡片，并加入低饱和度彩色线性图标。
- 服务器名称和地址并入右侧两行统计卡，彩色统计图标改为无底圈线性样式。

### 验证

```powershell
npm run check
```

## v0.92.91

- 移动端操作弹层新增顶部拖拽提示、主副标题层级和更清晰的按钮说明。
- 打开歌曲/播放操作弹层时锁定后台内容滚动，减少小屏误触和背景跟随滚动。
- 为操作弹层补充 Tab 焦点循环和辅助说明关联，键盘与读屏体验更稳定。
- 手机播放器“更多”里的队列、音质、播放模式、歌词和沉浸播放增加副说明，入口含义更明确。

### 验证

```powershell
npm run check
```

## v0.92.90

- 手机端登录页进一步压缩为单屏优先布局，顶部动态图标居中变紧凑，低高度屏幕自动减少非必要高度。
- 主页移动端统计卡改为横向滑动卡片，今日开播、标题区和操作按钮减少纵向占用。
- 手机底部播放器改为单行控制，进度条贴顶展示，并把播放模式放入“更多”操作弹层。
- 优化手机歌曲列表行、封面、音质标签和超窄屏时长显示，减少横向拥挤和误触。

### 验证

```powershell
npm run check
```

## v0.92.85

- 调整正在播放页唱片机唱臂：停靠位收回到唱片机右上边缘，播放落针幅度缩小，避免夸张摆动。
