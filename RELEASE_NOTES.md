## 版本说明

本版本重点修复音乐桥歌曲在重新进入应用后无法直接播放的问题。恢复队列、歌词读取和音质探测现在都会优先使用当前已连接的音乐桥地址，并保留可重新解析的插件歌曲快照，避免必须重新搜索才能播放。

## 更新内容

- 修复恢复队列中的音乐桥歌曲优先使用旧歌曲地址，导致重新进入后播放失败的问题。
- 播放、歌词读取、播放链路测试和搜索结果音质探测统一优先使用当前音乐桥地址，旧歌曲地址只作为兜底。
- 增强插件歌曲持久化快照，保留插件 URL、名称、来源 ID 和原始歌曲对象，便于重新解析播放地址。
- 音乐桥支持从持久化快照临时恢复插件运行时，即使插件 key 变化也能通过插件 URL 重新解析歌曲。
- 增加回归检查，覆盖恢复后的音乐桥歌曲无需重新搜索即可重新解析播放地址。
