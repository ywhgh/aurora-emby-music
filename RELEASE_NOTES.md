## 版本说明

本版本继续修复音乐桥插件音源“重新进入后不能播放”的核心问题。插件歌曲恢复时会保存更完整的插件身份信息，并在旧 `pluginKey` 失效时通过插件 URL 或名称重新匹配音乐桥插件，确保重新打开应用后也能实时向 `/media` 解析新的播放地址，不需要重新搜索歌曲才能播放。

## 更新内容

- 音乐桥插件歌曲持久化新增 `pluginUrl`、`pluginPlatform` 等恢复字段，队列和最近播放能保留更可靠的歌源身份。
- 前端播放成功后会把插件恢复字段同步回当前歌曲，下一次重进应用仍能拿到完整恢复快照。
- 兼容历史外部音源会话：旧记录只有 `serverUrl` 时，也能恢复音乐桥 API 地址。
- 音乐桥端恢复插件歌曲时不再只依赖旧 `pluginKey`，会按插件 URL、名称、平台兜底匹配并重新调用插件解析媒体。
- 修正歌词逐字高亮的浏览器检查，保持剪裁高亮走 `clip-path`，减少逐字播放时的布局抖动。
- 已验证旧插件 key 失效的恢复场景、真实音乐桥搜索与 `/media` 播放链路、桌面和手机浏览器 smoke。
