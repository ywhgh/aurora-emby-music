## 版本说明

本版本聚焦沉浸式播放页控制体验：将顶栏、空状态、队列抽屉和底部播放控制统一收敛为纯图标按钮，并补齐动态状态下的可访问标签和悬停提示。手机端和桌面端现在使用同一套沉浸式控制语义，避免文字按钮再次混入控制区造成视觉割裂或底部拥挤。

## 更新内容

- 沉浸式播放页顶栏的歌词校准、背景切换、全屏和退出按钮全部补齐图标按钮提示。
- 沉浸式空状态快捷操作、队列工具、底部播放模式、收藏、上一首、播放、下一首、纯享和队列按钮统一为纯图标表达。
- 新增图标按钮标签同步逻辑，播放/暂停、背景样式、纯享模式和队列开关会同步更新 `aria-label`、`title` 和读屏文本。
- 增加沉浸式按钮作用域 CSS 兜底，防止通用播放器按钮样式把文字标签重新带回沉浸式界面。
- 扩展桌面和手机真实 Chrome 冒烟测试，校验沉浸式动作按钮不显示文字、保留图标、可访问标签、悬停提示和可点击尺寸。
