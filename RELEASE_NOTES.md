# v0.92.92 Release Notes

本版本优化手机端搜索建议，并重排主页顶部数据区，让首页更紧凑、信息更集中。

## Highlights

- 搜索建议在手机端改为固定视口面板，减少裁切和溢出。
- 搜索结果保留快捷操作按钮，可直接播放、下一首播放或加入队列。
- 搜索建议行距、封面和图标尺寸更适合小屏操作。
- 主页媒体统计移动到欢迎区右侧，减少首屏纵向占用。
- 统计卡升级为低饱和度彩色线性图标和紧凑数据层级。
- 服务器名称和地址并入两行统计区，图标改为更大的彩色线条。
