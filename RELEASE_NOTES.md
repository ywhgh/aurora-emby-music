## 版本说明

本版本继续优化沉浸播放页的逐字歌词流畅度。逐字进度写入现在会统一规范化，减少浮点抖动带来的无意义样式更新；当沉浸歌词的 RAF 循环已经接管播放进度时，粗粒度 `timeupdate` 不再重复驱动歌词高亮，降低播放中歌词节奏不均的概率。

## 更新内容

- 逐字歌词进度统一保留 `0.1%` 精度并格式化为稳定百分比字符串，减少热路径 CSS 写入抖动。
- 沉浸歌词 RAF 循环运行时，播放进度 `timeupdate` 只校准平滑歌词时钟，不再重复写入逐字高亮。
- 保留 seek、暂停恢复和普通播放页的强制歌词刷新逻辑，避免影响跳转后的即时同步。
- 扩展密集歌词 browser smoke，验证 CSS 进度写入格式、RAF 接管时的 `timeupdate` 写入抑制，以及桌面/移动端稳定性。
- 更新缓存版本到 `0.93.120`，确保浏览器加载包含歌词流畅度修复的新脚本。
