## 版本说明

本版本修复音乐桥插件歌曲在重新进入软件或音乐桥重启后无法直接播放的问题。音乐桥现在会持久化插件歌曲恢复信息，并在只剩 `plugin:key:songId` 的旧队列状态下自动重建播放请求，不再需要先重新搜索一次才能播放。

## 更新内容

- 音乐桥新增插件歌曲本地缓存，搜索过的插件歌曲会保存恢复快照，桥端重启后仍可直接解析播放。
- `/media` 增加插件 ID 兜底恢复：只传 `plugin:key:songId` 时也会自动重建插件 track，避免返回 404。
- 前端补齐旧队列/旧缓存歌曲的最小恢复快照，缺少完整 raw 数据时仍会带上可恢复的歌曲 ID 和标题信息。
- 修复重新进入软件后音乐桥歌曲必须重新搜索才能播放的问题，恢复播放会强制获取实时播放地址。
- 优化沉浸歌词逐字高亮的合成提示，只对当前播放行启用 `clip-path` 动画提示，降低长歌词列表的渲染压力。
- 更新缓存版本到 `0.93.125`，确保浏览器加载新的音乐桥恢复逻辑。
